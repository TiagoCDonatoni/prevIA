
Coloque aqui o Python portátil (embeddable).

Passo a passo sugerido (Windows 64 bits):

1. Acesse:
   https://www.python.org/downloads/windows/

2. Na seção "Stable Releases", procure por:
   - "Windows embeddable package (64-bit)"

3. Baixe o arquivo ZIP (por exemplo: python-3.12.0-embed-amd64.zip).

4. Extraia TODO o conteúdo desse ZIP diretamente para esta pasta:
   - python-portable\

   Depois de extrair, você deve ver, por exemplo:
   - python-portable\python.exe
   - python-portable\python3.dll
   - etc.

5. Com isso pronto, é só voltar para a raiz do projeto e dar duplo clique em:
   - rodar.bat
